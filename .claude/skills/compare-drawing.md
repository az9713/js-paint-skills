# Drawing Comparison & Iteration Skill

## Purpose

This skill guides the iterative improvement loop: draw, screenshot, evaluate, fix, repeat. It's the "automated test criteria" that Kris referenced from Mo's evolution analogy.

## Workflow

### 1. Take a Screenshot

After completing a drawing phase, capture the current state:

```
Use computer tool with action: "screenshot"
```

### 2. Evaluate the Drawing

When viewing the screenshot, assess these criteria:

**Composition Check:**
- Is the subject centered/positioned well?
- Are proportions reasonable?
- Is the background filled appropriately?

**Color Check:**
- Are colors appropriate for the scene?
- Is there enough color variety/contrast?
- Do colors match the intended mood?

**Detail Check:**
- Are key features visible? (eyes, limbs, defining characteristics)
- Are there any glaring errors? (backwards text, missing limbs, wrong proportions)
- Is there enough detail for the style? (oil painting = loose strokes OK, portrait = need detail)

**Technical Check:**
- Are there unintended gaps or white spots?
- Did strokes render properly on the canvas?
- Is the drawing contained within the canvas?

### 3. Prioritize Fixes

Rank issues by impact:
1. **Critical** - Missing major elements (no eyes on portrait, no legs on dog)
2. **High** - Wrong proportions or colors
3. **Medium** - Missing details (highlights, texture, small features)
4. **Low** - Minor imperfections (stroke overlap, slight asymmetry)

### 4. Apply Fixes

For each issue:
1. Select the appropriate tool and color
2. Draw the fix using the drawing helpers
3. Take another screenshot to verify

### 5. Iteration Criteria

**Stop when:**
- All critical and high issues are resolved
- The drawing is recognizable as the intended subject
- 2-3 iteration rounds have been completed
- Adding more detail starts degrading quality (diminishing returns)

**Keep going when:**
- Major features are missing
- Colors are wrong
- The subject is unrecognizable
- There's a specific fix that would significantly improve quality

## Comparison Against Reference Image

When comparing to a reference image (like fisherman.png):

1. **Read the reference** with the Read tool first
2. **Take screenshot** of current canvas
3. **Compare side by side** mentally:
   - Shape similarity
   - Color accuracy
   - Position/layout match
   - Detail level match
4. **List differences** explicitly
5. **Fix the biggest difference first**

## Quick Evaluation Checklist

Use this after each screenshot:

- [ ] Subject is recognizable
- [ ] Colors are appropriate
- [ ] Key features present (eyes, limbs, etc.)
- [ ] Background is filled
- [ ] No major technical glitches
- [ ] Overall composition is balanced

## Tips

- Don't aim for perfection - aim for recognizable and stylistically consistent
- Oil paintings should look loose and impressionistic - too much detail ruins the style
- Pencil portraits need more precision on facial features
- After 3 iterations, stop. Diminishing returns are real.
- Screenshot at EVERY phase transition, not just at the end
